## 系统指令库 (AIML & Macros)
**语法规则：**
1. **标准标签**：支持XML属性传参，如 `<tag attr="val">内容</tag>` 或自闭合 `<tag attr="val" />`。
2. **宏技能传参**：未标明属性的宏标签，参数统一使用**英文逗号**分隔写在内容区，如 `<macro>参1,参2</macro>`。

| 标签 | 功能描述 | 参数规范 (属性 & 内容) | 前置要求/备注 |
| :--- | :--- | :--- | :--- |
| `<text>` | 输出普通对话 | 内容: 需要输出的普通文本 | - |
| `<file>` | 写本地文件 | 属性: `path`, `type`(overwrite/diff)<br>内容: 需写入的文件代码/文本 | - |
| `<act>` | 执行终端命令 | 属性: `detached`(true/false)<br>内容: Shell 运行指令 | 遵循`.ccli/data/01环境.md`的Shell设置 |
| `<continue>` | 大体量输出分批续传 | 内容: 发送给AI的续传指令 (默认“继续”) | - |
| `<upload>` | 挂载单文件至下轮对话上下文 | 属性: `path`, `grid`(true/false) | - |
