## 系统指令库 (AIML & Macros)

| 标签 | 功能描述 | 内容 | 属性 | 前置要求 |
| :--- | :--- | :--- | :--- | :--- |
| `<text>` | 输出普通对话 | 需要输出的普通文本 | - | - |
| `<file>` | 读写、搜索本地文件与挂载附件,附件仅支持二进制 | `write`或`diff`时填文本内容，`diff`填标准 SEARCH/REPLACE 块  | `action`(write/diff/search/tree/pack/upload), `path`, `keyword`(search时必填) | - |
| `<shell>` | 执行终端命令或程序 | Shell 运行指令或可执行文件绝对路径 | `mode`(sync/detached/launch,默认sync) | - |
| `<clipboard>` | 读写系统剪贴板 | `action="write"`时填写文本 | `action`(read/write) | - |
| `<continue>` | 支持下次自动调用AI | 回调AI内容，例如：“继续” | - | - |
| `<window>` | 管理操作系统窗口生命周期与物理属性提取 | - | `action`(activate/close/min/max/restore/info), `target`(窗口标题或进程名) | - |
| `<vision>` | 视觉感知，支持全屏截图与本地 OCR | - | `action`(screenshot/grid/ocr), `model`(tesseract/PaddleOCR) | - |
| `<mouse>` | 执行基于屏幕绝对坐标的物理鼠标事件 | - | `action`(click/double_click/drag/hover/scroll), `x`, `y`, `btn`, `x1`, `y1`, `x2`, `y2`, `dir`(up/down/left/right), `amount` | 极度依赖视觉或窗口数据，严禁凭空猜测坐标 |
| `<keyboard>` | 原样输入长文本代码或触发物理快捷键 | `action="text"`填普通文本；`action="type"`填按键表达式(如 `^c`, `{Enter}`)，两者严禁混用 | `action`(text/type) | 输入前确保目标输入框已激活 |
| `<browser>` | 浏览器自动化综合控制 | 目标URL或需要填入的文本值 | `action`(goto/scan/read/click/fill/select), `url`, `id`, `value` | 点击/输入前必须先执行scan获取id |
| `<uia>` | 桌面原生UI自动化控制（基于UIA） | 仅fill时可填入文本值 | `action`(scan/click/fill/scroll), `target`(窗口名或类名), `id`, `value` | 点击/输入前必须先执行scan获取id |
| `<wait>` | 全系统通用流控等待 | - | `type`(sleep/networkidle/dom/window), `timeout`, `target` | - |
| `<assert>` | 状态断言与强制校验，条件不满足会抛错打断 | - | `type`(ui_exist/window_active/text_exist), `target`, `timeout` | - |
| `<os>` | 操作系统外围控制 | - | `action`(mute/volume_up/volume_down/lock/empty_trash) | - |
| `<listen>` | 异步事件监听与守护 | 监控目标与回调指令 | `type`(file/process/webhook), `target`, `keyword`, `callback_prompt` | - |
| `<network>` | 原生 HTTP/API 通信 | 请求体或附加数据内容 | `action`(get/post/put/delete/download/upload), `url`, `headers`, `save_path` | - |
| `<device>` | 跨端物理设备控制 | 具体发送的键值或终端指令 | `action`(connect/tap/swipe/keyevent/shell), `target`, `x`, `y`, `key` | 需设备开启 ADB 调试端口 |
| `<vault>` | 物理注入环境凭证密钥，确保密码不在上下文明文出现 | - | `action`(type/clip), `key`(凭证代号标识) | - |
| `<db>` | 数据库原生直连，将海量结果转为JSON或落地为文件 | 标准 SQL 查询语句 | `type`(mysql/pg/sqlite), `dsn_key`, `save_path` | 需在配置中预置对应的 dsn_key |
| `<sandbox>`| 本地代码安全沙箱，进行计算下推与数据清洗 | 合法的、可独立运行的脚本代码 | `lang`(node/python) | - |
| `<stream>` | 截获底层音频与流媒体通道进行实时通信 | `speak` 模式填合成的文本内容 | `action`(listen/speak/stop), `device` | - |
| `<human>` | 物理世界交互与人机兜底，挂起任务等待外部信号 | 需要发送给人类的提示语或问题 | `type`(approval/qa/rescue/physical), `channel`, `target`, `hardware`, `timeout` | - |
| `<captcha>` | 专门处理验证码对抗与风控阻断 | `manual`时填提示语 | `type`或`solver`(auto/manual) | - |
| `<memory>` | 本地语义经验与企业偏好记忆的读写 | `store`或`recall`的具体内容/检索词 | `action`(store/recall), `domain` | - |
