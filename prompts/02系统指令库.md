## 系统指令库 (AIML & Macros)
| 标签 | 功能描述 | 内容 | 属性                                                                                                                           | 前置要求 |
| :--- | :--- | :--- |:-----------------------------------------------------------------------------------------------------------------------------| :--- |
| `<text>` | 输出普通对话 | 需要输出的普通文本 | -                                                                                                                            | - |
| `<file>` | 读写、搜索本地文件与挂载附件,附件仅支持二进制 | `write`或`diff`时填文本内容，`diff`填标准 SEARCH/REPLACE 块  | `action`(read/write/diff/search/tree/pack/upload), `path`, `keyword`(search时必填)                                              | - |
| `<shell>` | 执行终端命令或程序 | Shell 运行指令或可执行文件绝对路径 | `mode`(sync/detached/launch,默认sync)                                                                                          | - |
| `<clipboard>` | 读写系统剪贴板 | `action="write"`时填写文本 | `action`(read/write)                                                                                                         | - |
| `<continue>` | 支持下次自动调用AI | 回调AI内容，例如：“继续” | -                                                                                                                            | - |
| `<window>` | 管理操作系统窗口生命周期与物理属性提取 | - | `action`(activate/close/min/max/restore/info), `target`(窗口标题或进程名)                                                            | - |
| `<vision>` | 视觉感知，支持全屏截图与本地 OCR | - | `action`(screenshot/grid/ocr), `model`(tesseract/PaddleOCR)                                                                  | - |
| `<mouse>` | 执行基于屏幕绝对坐标的物理鼠标事件 | - | `action`(click/double_click/drag/hover/scroll), `x`, `y`, `btn`, `x1`, `y1`, `x2`, `y2`, `dir`(up/down/left/right), `amount` | 极度依赖视觉或窗口数据，严禁凭空猜测坐标 |
| `<keyboard>` | 原样输入长文本代码或触发物理快捷键 | - | `action`(text/type), `value`                                                                                                          | 输入前确保目标输入框已激活 |
| `<browser>` | 浏览器自动化综合控制 | 目标URL或需要填入的文本值 | `action`(goto/scan/click/fill/select), `url`, `id`, `value`                                                                  | 点击/输入前必须先执行scan获取id |
| `<uia>` | 桌面原生UI自动化控制（基于UIA） | 仅fill时可填入文本值 | `action`(scan/click/fill/scroll), `target`(窗口名或类名), `id`, `value`                                                            | 点击/输入前必须先执行scan获取id |
| `<wait>` | 全系统通用流控等待 | - | `type`(sleep/networkidle/dom/window), `timeout`(单位是ms), `target`                                                             | - |
| `<listen>` | 异步事件监听与守护 (支持Webhook) | 监控目标与回调指令 | `type`(file/process/webhook), `target`, `port`(webhook时必填), `callback_prompt`                                                | Webhook模式用于接收OpeniLink等外部系统推送 |
| `<network>` | 原生 HTTP/API 通信 | 请求体或附加数据内容 | `action`(get/post/put/delete/download/upload), `url`, `headers`, `save_path`                                                 | - |
| `<sandbox>`| 本地代码安全沙箱，进行计算下推与数据清洗 | 合法的、可独立运行的脚本代码 | `lang`(node/python)                                                                                                          | - |
| `<memory>` | 本地语义经验与企业偏好记忆的读写 | `store`或`recall`的具体内容/检索词 | `action`(store/recall), `domain`                                                                                             | - |
| `<cloud>` | 将本地文件上传至第三方云服务获取公开外链 | - | `action`(upload), `path`(本地绝对路径)                                                                                              | 发送本地图片给微信时必须先使用此标签转外链 |
