## 系统指令库 (AIML & Macros)
| 标签 | 功能描述 | 内容                    | 属性                                                                                                 | 前置要求 |
| :--- | :--- |:----------------------|:-----------------------------------------------------------------------------------------------------------------------------| :--- |
| `<text>` | 输出普通对话 | 需要输出的普通文本             | -                                                                                                                            | - |
| `<file>` | 读写、搜索本地文件与挂载附件,附件仅支持二进制 | 文本内容                  | `action`(read/write/search/tree/pack/upload), `path`, `keyword`(search时必填)                                                   | - |
| `<shell>` | 执行终端命令或程序 | Shell 运行指令或可执行文件绝对路径,使用launch  | `mode`(sync/detached/launch,默认sync)                                                                                          | - |
| `<continue>` | 支持下次自动调用AI | 回调AI内容，例如：“继续”        | -                                                                                                                            | - |
| `<vision>` | 截图或截图加网格 | -                     | `action`(screenshot/grid)                                                               | - |
| `<browser>` | 浏览器自动化综合控制 | 目标URL或需要填入的文本值        | `action`(goto/scan/click/fill/select), `url`, `id`, `value`                                                                  | 点击/输入前必须先执行scan获取id |
| `<uia>` | 桌面原生UI自动化控制| 仅fill时可填入文本值          | `action`(scan/click/fill/scroll), `target`(窗口名或类名), `id`, `value`                                                            | 点击/输入前必须先执行scan获取id |
| `<wait>` | 全系统通用流控等待 | -                     | `timeout`(单位是ms)                                                           | - |
| `<memory>` | 本地语义经验与企业偏好记忆的读写 | 文本内容                  | `action`(store/recall), `domain`                                                                                             | - |
| `<weixin>` | 物理后台推送微信文本或本地图片 | 文本内容或图片附言 | `action`(send/send_image), `path`(send_image时必填的本地路径)                                                                                             | - |
