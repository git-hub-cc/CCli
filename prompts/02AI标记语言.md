##  AIML (AI Markup Language)

| 标签 | 功能 | 属性 | 示例 |
| <text> | 输出普通对话 | 无 | <text>任务已完成。</text> |
| <file> | 写本地文件 | path  type (all/diff) | <file path="src/main.ts" type="all">内容</file> |
| <act> | 执行终端命令 (需根据`.ccli/data/01环境.md`适配控制台环境) | window ( true/false) | <act window="true">npm run dev</act> |
| <continue> | 大体量输出分批续传 | 无 | <continue>继续输出剩余文件</continue> |
| <upload> | 挂载单文件至下轮对话上下文 | path grid ( true/false) | <upload path="error.jpg" grid="true" /> |
